# Nesta Coffee - E-Commerce Website

## Features

- **Responsive Layout**: Seamlessly adapts to all screen sizes.
- **Order Placement**: Add coffee to cart and proceed to checkout.
- **Order Confirmation**: Review and confirm your order.
- **Cart Management**: View, update, and remove items.
- **Location Services**: Confirm delivery location.
- **User Authentication**: Secure login and profile management.
- **Local Storage**: Persistent user data across sessions.
- **Modern UI/UX**: Smooth animations, optimized performance, and accessibility.

## Technologies Used

- **React**: For building the user interface.
- **Tailwind CSS**: For responsive and utility-first styling.
- **Local Storage**: To persist user and cart data.
- **JavaScript (ES6+)**: For dynamic functionality.
